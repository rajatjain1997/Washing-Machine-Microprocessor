# Washing-Machine-Microprcessor

## Addresses
* _Memory_
	* ROM: 00000h - 01FFFh
	* RAM: 02000h - 02FFFh
* _IO_
	* 8255:	00h - 07h

## Interfacing
* _8255_
	* Port A:
		1. Start
		2. Stop
		3. Load
		4. Resume
		5. Door Lock
		6. Water Max
		7. Water Min
	* Port B: 
		1. Agitator
		2. Wash Tub
		3. Buzzer 3
		4. Buzzer 2
		5. Buzzer 1
	* Port C:
		1. Seven Seg Display
		2. Seven Seg Display
		3. Seven Seg Display
		4. Seven Seg Display
		5. Gate of counter 0
		6. Gate of counter 1
		7. Gate of counter 3
